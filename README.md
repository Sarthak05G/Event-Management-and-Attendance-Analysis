📌 Project Overview

This project focuses on analyzing event performance and attendee behavior using Power BI.
The dashboard provides insights into event attendance, registrations, engagement trends, revenue performance, and attendee demographics to help organizers make data-driven decisions.

The project demonstrates skills in:

Data Cleaning & Transformation
Data Modeling
DAX Calculations
Interactive Dashboard Design
Business Insights Generation
🚀 Objectives
Track total event registrations and attendance
Analyze attendance trends across different events
Measure event engagement and participation rates
Identify high-performing event categories
Monitor revenue generated from events
Understand attendee demographics and behavior
🛠️ Tools & Technologies
Microsoft Power BI
Power Query
DAX (Data Analysis Expressions)
Excel / CSV Dataset
